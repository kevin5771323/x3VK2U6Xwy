# JAVA酒店管理系统 java498

本项目是基于Springboot和Vue的酒店管理系统，适用于项目实训和在线课程任务。以下是系统的相关介绍。

## 技术栈

- 前端框架：Vue
- 后端框架：Springboot

## 运行环境

- JDK 1.8
- MySQL 5.7/8.0
- Maven 3.5
- Node.js 16.20
- 开发工具：IDEA、VSCode、MySQLWorkbench/Navicat

## 系统角色与功能模块

### 用户功能模块

- 登录
- 注册
- 查看系统公告
- 留言
- 预约入住
- 修改个人信息

### 管理员功能模块

- 登录
- 管理员管理
- 用户管理
- 房型管理
- 房间管理
- 预约管理
- 入住管理
- 留言管理
- 公告管理

## 部署步骤

1. 准备运行环境，包括JDK、MySQL、Maven等。
2. 搭建项目所需的后端服务，如MySQL数据库。
3. 使用Maven或IDEA导入项目，并编译。
4. 运行前端项目，配置好与后端服务的连接。
5. 启动后端服务，访问前端项目进行测试。
